## Requirements
- PHP 5.6+
- concrete5 8.1+

## Installation

To install this package on a composer based concrete5 site:
```bash
$ composer require a3020/composer_security
```

Then install the package:
```bash
$ ./vendor/bin/concrete5 c5:package-install composer_security
```
